Expand description

Enriched event ready for UI display, SIEM export, and policy evaluation.

Produced by the agent from raw eBPF events after classification, ML analysis, and container/Kubernetes enrichment. Serialized as NDJSON over the Unix socket to the UI and SIEM sinks.
