Crate busted_theme 

Source
Expand description

Shared color theme for busted dashboards (TUI + GUI).

All colors are defined as (u8, u8, u8) RGB tuples so that framework-specific crates (ratatui, egui) can convert them to their own color types without pulling in either dependency.
