Text embedding for AI agent identity fingerprinting.

Provides a simple API for generating text embeddings using sentence-transformers/all-MiniLM-L6-v2 (384-dimensional output).

Without the candle feature, the crate compiles but Embedder::new() returns an error.

use busted_embed::Embedder;

assert_eq!(embedding.len(), 384);
