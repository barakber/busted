1//! Identifies LLM API traffic from network behavior alone — no payload inspection
2//! required.
3//!
4//! # Why behavioral classification?
5//!
6//! Content-based classification (`busted-classifier`) works
7//! beautifully when you can read the payload: intercept TLS via eBPF uprobes, parse
8//! the HTTP, match the endpoint, done. But sometimes you *can't* read the payload.
9//! The application uses certificate pinning. The traffic is multiplexed over HTTP/2
10//! and you only see opaque frames. The process links a TLS library you don't have
11//! uprobes for. In all of these cases, content classification is blind.
12//!
13//! `busted-ml` takes a completely different approach. Instead of looking at *what* a
14//! process says, it watches *how* it talks to the network — timing between packets,
15//! sizes, bursts, port diversity — and learns to recognize LLM traffic from its
16//! behavioral signature.
17//!
18//! # The insight
19//!
20//! LLM API calls have a distinctive network fingerprint. A typical interaction looks
21//! like: large POST request (the prompt), long pause (model inference), then a burst
22//! of small chunked responses (token streaming). This pattern — big-send, wait,
23//! trickle-back — is very different from web browsing (many small GETs, fast
24//! responses), file downloads (one request, sustained transfer), or database traffic
25//! (rapid small round-trips). The ML model learns these signatures automatically from
26//! labeled examples.
27//!
28//! # How it works
29//!
30//! Each network event arriving from the eBPF layer gets compressed into one of 180
31//! discrete symbols — encoding the event type (connect, send, receive, close), a
32//! byte-size bucket, and a port class. These symbols accumulate in a per-process
33//! sliding window.
34//!
35//! When the window fills, a 352-dimensional feature vector is extracted: unigram,
36//! bigram, and trigram frequency histograms capture *what* happened; timing statistics
37//! and burst analysis capture *when*; connection diversity and entropy measures capture
38//! *where*. This feature vector is the input to two parallel classifiers.
39//!
40//! The **supervised path** feeds the features into a bagged ensemble of 100 decision
41//! trees (a random forest built on [`linfa`]). The forest is trained online — as new
42//! events arrive with IP-based ground truth labels from the agent, the model
43//! incrementally improves. It outputs a provider label and confidence score.
44//!
45//! The **unsupervised path** feeds the same features into [`hdbscan`] density-based
46//! clustering. This catches novel traffic patterns that the supervised model hasn't
47//! seen — a new LLM provider, an unusual SDK, or a tool using AI in an unexpected
48//! way. Novel clusters surface as `is_novel: true` in the result.
49//!
50//! # Architecture
51//!
52//! The classifier operates on a per-PID sliding window of [`busted_types::NetworkEvent`]s:
53//!
54//! 1. **Symbolization** — Each event is mapped to a compact symbol encoding event type,
55//!    byte-size bucket, and port class (180 possible symbols).
56//! 2. **Feature extraction** — A 352-dimensional feature vector is computed from unigram,
57//!    bigram, and trigram histograms, timing statistics, byte patterns, connection
58//!    diversity, burst analysis, and entropy measures.
59//! 3. **Supervised classification** — A bagged ensemble of 100 decision trees (random
60//!    forest via [`linfa`]) is trained online from IP-labeled ground truth.
61//! 4. **Unsupervised discovery** — [`hdbscan`] clustering detects novel traffic patterns
62//!    not captured by the supervised model.
63//!
64//! # Integration with Busted
65//!
66//! In the Busted agent, `busted-ml` sits behind the `ml` feature flag. The agent feeds
67//! every [`NetworkEvent`] to [`MlClassifier::process_event`],
68//! along with any IP-based provider label it already has. The returned
69//! [`BehaviorIdentity`] is folded into the
70//! `BustedEvent` as `ml_confidence`,
71//! `ml_provider`, `behavior_class`, and `cluster_id` fields. This means the UI and SIEM
72//! sinks get both content-based *and* behavioral classifications for every connection.
73//!
74//! # Usage
75//!
76//! ```no_run
77//! use busted_ml::MlClassifier;
78//! use std::time::Duration;
79//!
80//! let mut classifier = MlClassifier::new();
81//!
82//! // Feed network events as they arrive:
83//! // let result = classifier.process_event(&event, Some("OpenAI"));
84//! // if let Some(identity) = result {
85//! //     println!("class={}, confidence={}", identity.class, identity.confidence);
86//! // }
87//!
88//! // Periodically garbage-collect idle PID windows:
89//! classifier.gc_idle_pids(Duration::from_secs(300));
90//! ```
91//!
92//! # Key Types
93//!
94//! | Type | Description |
95//! |------|-------------|
96//! | [`MlClassifier`] | Top-level coordinator — owns windows, classifier, and discovery |
97//! | [`BehaviorClass`] | Classification result enum (`LlmApi`, `GenericHttps`, `DnsHeavy`, `Unknown`) |
98//! | [`BehaviorIdentity`] | Full result with class, confidence, cluster ID, and signature |

119/// High-level behavioral classification result.
120#[derive(Clone, Debug, Serialize)]
121#[allow(dead_code)]
122pub enum BehaviorClass {
123    /// Traffic pattern matches a known LLM API provider.
124    LlmApi(String),
125    /// Generic HTTPS traffic (port 443, no LLM indicators).
126    GenericHttps,
127    /// DNS-heavy traffic pattern.
128    DnsHeavy,
129    /// Could not classify the traffic pattern.
130    Unknown,
131}
132
133impl std::fmt::Display for BehaviorClass {
134    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
135        match self {
136            BehaviorClass::LlmApi(provider) => write!(f, "LlmApi({})", provider),
137            BehaviorClass::GenericHttps => write!(f, "GenericHttps"),
138            BehaviorClass::DnsHeavy => write!(f, "DnsHeavy"),
139            BehaviorClass::Unknown => write!(f, "Unknown"),
140        }
141    }
142}

144/// Full behavioral identity for a traffic window.
145#[derive(Clone, Debug, Serialize)]
146pub struct BehaviorIdentity {
147    /// Predicted traffic class.
148    pub class: BehaviorClass,
149    /// Classifier confidence (0.0–1.0).
150    pub confidence: f64,
151    /// HDBSCAN cluster ID (-1 = noise / not yet clustered).
152    pub cluster_id: i32,
153    /// Hash of top-5 bigrams as a behavioral signature.
154    pub signature: u64,
155    /// Number of events in the window when classified.
156    pub window_size: usize,
157    /// True if the pattern doesn't match any known provider.
158    pub is_novel: bool,
159}

161/// Top-level ML classifier coordinator.
162pub struct MlClassifier {
163    windows: HashMap<u32, EventWindow>,
164    extractor: FeatureExtractor,
165    classifier: TrainedClassifier,
166    discovery: PatternDiscovery,
167    training_buffer: Vec<(Array1<f64>, String)>,
168    config: WindowConfig,
169    samples_since_train: usize,
170    retrain_interval: usize,
171    max_training_buffer: usize,
172}
173
174impl Default for MlClassifier {
175    fn default() -> Self {
176        Self::new()
177    }
178}

180impl MlClassifier {
181    pub fn new() -> Self {
182        MlClassifier {
183            windows: HashMap::new(),
184            extractor: FeatureExtractor::new(),
185            classifier: TrainedClassifier::new(),
186            discovery: PatternDiscovery::new(),
187            training_buffer: Vec::new(),
188            config: WindowConfig::default(),
189            samples_since_train: 0,
190            retrain_interval: 200,
191            max_training_buffer: 10_000,
192        }
193    }
194
195    /// Process a single network event. `ip_label` is the provider string from
196    /// IP-based classification (e.g. "OpenAI"), or None if unclassified.
197    /// Returns a BehaviorIdentity when the window is ready for classification.
198    pub fn process_event(
199        &mut self,
200        event: &NetworkEvent,
201        ip_label: Option<&str>,
202    ) -> Option<BehaviorIdentity> {
203        let symbol = Symbol::from_network_event(event);
204
205        // FNV-1a hash of destination IP bytes for diversity tracking
206        let dst_ip_hash = {
207            let bytes = unsafe { event.daddr.ipv6 };
208            let mut h: u32 = 0x811c9dc5;
209            for &b in &bytes {
210                h ^= b as u32;
211                h = h.wrapping_mul(0x01000193);
212            }
213            h
214        };
215
216        let timed = TimedSymbol {
217            symbol,
218            timestamp_ns: event.timestamp_ns,
219            bytes: event.bytes,
220            arrived_at: Instant::now(),
221            dst_ip_hash,
222            dst_port: event.dport,
223        };
224
225        let window = self
226            .windows
227            .entry(event.pid)
228            .or_insert_with(|| EventWindow::new(event.pid));
229
230        let ready = window.push(timed, &self.config);
231        if !ready {
232            return None;
233        }
234
235        // Extract features
236        let features = self.extractor.extract(window);
237        let window_size = window.len();
238        window.mark_classified();
239
240        // Feed training buffer if we have a ground truth label
241        let label = if let Some(provider) = ip_label {
242            provider.to_string()
243        } else if event.dport == 443 {
244            "GenericHttps".to_string()
245        } else {
246            "Unknown".to_string()
247        };
248
249        self.training_buffer.push((features.clone(), label.clone()));
250        self.samples_since_train += 1;
251
252        // FIFO eviction
253        if self.training_buffer.len() > self.max_training_buffer {
254            let excess = self.training_buffer.len() - self.max_training_buffer;
255            self.training_buffer.drain(0..excess);
256        }
257
258        // Retrain periodically
259        if self.samples_since_train >= self.retrain_interval {
260            self.samples_since_train = 0;
261            match self.classifier.train(&self.training_buffer) {
262                Ok(()) => {
263                    log::info!("ML model trained on {} samples", self.training_buffer.len());
264                }
265                Err(e) => {
266                    log::debug!("ML training skipped: {}", e);
267                }
268            }
269        }
270
271        // Classify
272        let (class, confidence, is_novel) =
273            if let Some((pred_label, conf)) = self.classifier.predict(&features) {
274                let class = match pred_label.as_str() {
275                    "GenericHttps" => BehaviorClass::GenericHttps,
276                    "Unknown" => BehaviorClass::Unknown,
277                    provider => BehaviorClass::LlmApi(provider.to_string()),
278                };
279                let is_novel = ip_label.is_none() && matches!(&class, BehaviorClass::LlmApi(_));
280                (class, conf, is_novel)
281            } else {
282                (BehaviorClass::Unknown, 0.0, false)
283            };
284
285        // Unsupervised discovery
286        let cluster_id = self.discovery.ingest(features.as_slice().unwrap());
287
288        // Compute behavioral signature from top-5 bigram bins
289        let signature = compute_signature(features.as_slice().unwrap());
290
291        Some(BehaviorIdentity {
292            class,
293            confidence,
294            cluster_id,
295            signature,
296            window_size,
297            is_novel,
298        })
299    }
300
301    /// Remove windows for PIDs that have been idle longer than `max_idle`.
302    pub fn gc_idle_pids(&mut self, max_idle: Duration) {
303        let now = Instant::now();
304        self.windows.retain(|_, w| {
305            w.last_arrival()
306                .is_some_and(|t| now.duration_since(t) < max_idle)
307        });
308    }
309}

374/// Hash the top-5 bigram bin values as a behavioral signature.
375fn compute_signature(features: &[f64]) -> u64 {
376    let bigram_start = symbol::SYMBOL_SPACE;
377    let bigram_end = bigram_start + 100;
378
379    if features.len() < bigram_end {
380        return 0;
381    }
382
383    // Find top-5 bigram bins by value
384    let mut indexed: Vec<(usize, f64)> = features[bigram_start..bigram_end]
385        .iter()
386        .enumerate()
387        .map(|(i, &v)| (i, v))
388        .collect();
389    indexed.sort_by(|a, b| b.1.partial_cmp(&a.1).unwrap_or(std::cmp::Ordering::Equal));
390
391    let mut h: u64 = 0xcbf29ce484222325;
392    for &(idx, _) in indexed.iter().take(5) {
393        h ^= idx as u64;
394        h = h.wrapping_mul(0x100000001b3);
395    }
396    h
