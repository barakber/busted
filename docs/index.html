Features

Kernel-native observability and policy enforcement for LLM/AI traffic

K

Kernel-Native Monitoring

eBPF kprobes, uprobes, and LSM hooks with minimal overhead. Events flow through a 512KB RingBuf with zero copies.

T

TLS Plaintext Capture

Intercepts decrypted data from OpenSSL SSL_write/SSL_read. See actual LLM prompts and responses in cleartext.

L

LLM/MCP Protocol Detection

Identifies API calls to 15+ LLM providers and MCP JSON-RPC tool invocations using content analysis and SNI matching.

P

OPA Policy Enforcement

Evaluate events against Rego policies for allow, audit, or deny decisions. Kernel-level enforcement via LSM + SIGKILL.

I

Agent Identity Tracking

Correlates weak per-event signals into stable agent identities. Builds action timelines and narrative descriptions.

M

ML Behavioral Classification

Decision trees and HDBSCAN clustering detect LLM traffic patterns from network behavior alone when payloads are unavailable.

Demo

Real-time policy enforcement blocking AI agent traffic

sudo busted monitor --enforce --rule '
  package busted
  default decision = "allow"
  decision = "deny" {
      input.llm_provider == "Anthropic"
      contains(input.tls_payload, "berko")
  }'

Quick Start

Three ways to get running

APT (Debian/Ubuntu)

echo "deb [trusted=yes] https://barakber.github.io/busted/repos/apt ./" \
  | sudo tee /etc/apt/sources.list.d/busted.list
sudo apt-get update && sudo apt-get install busted
sudo busted monitor

From Source

rustup toolchain install nightly --component rust-src
cargo install bpf-linker
git clone https://github.com/barakber/busted && cd busted
cargo xtask build --features tls
sudo ./target/debug/busted monitor --verbose

Docker

docker build -f deploy/Dockerfile -t busted:latest .
docker run --privileged --pid=host -v /sys:/sys:ro busted:latest

Crate Documentation

Rustdoc for every crate in the workspace

busted_types

The data contract between kernel and userspace. Flat, #[repr(C)], fixed-size types that travel through the eBPF ring buffer with zero copies.

busted_classifier

Stateless content classification for decrypted TLS payloads. Identifies LLM API calls, MCP JSON-RPC messages, SDK fingerprints, and PII.

busted_identity

Cross-event AI agent identity resolution. Correlates weak signals (PID, SDK, model, fingerprint) into stable identities with action timelines.

busted_ml

Behavioral traffic classifier that identifies LLM API traffic from network patterns alone — packet timing, sizes, bursts — when payload inspection is unavailable.

busted_opa

OPA/Rego policy engine for LLM communication governance. Evaluates events against user-defined Rego policies and returns allow, audit, or deny decisions.

busted_agent

Userspace eBPF monitoring agent. Loads kernel probes, consumes events, classifies traffic, evaluates policies, and broadcasts to CLI/UI/SIEM.

busted_ui

Native egui dashboard that connects to the agent over a Unix socket and displays live classified events with provider stats and identity columns.

busted (CLI)

Unified CLI binary: busted monitor, busted policy {check,eval,test}, busted ui. The canonical cargo install target.

busted_ebpf

eBPF kernel programs: kprobes for TCP/UDP, uprobes for OpenSSL, LSM hooks for policy enforcement. Built for bpfel-unknown-none.

xtask

Build automation: cargo xtask build compiles eBPF programs and the full workspace. Also handles version bumps and consistency checks.
