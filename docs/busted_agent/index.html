Source
Expand description

Userspace eBPF monitoring agent for LLM/AI communication tracking.

This crate loads eBPF programs into the kernel (kprobes, uprobes, LSM hooks), consumes events via a RingBuf, classifies traffic against known LLM providers, and broadcasts [BustedEvent]s to CLI output, a Unix socket server (for the dashboard UI), and optional SIEM sinks.

§Usage

use busted_agent::{AgentConfig, run_agent};

let config = AgentConfig {
    verbose: false,
    format: "text".into(),
    enforce: false,
    output: "stdout".into(),
    policy_dir: None,
    policy_rule: None,
    metrics_port: 9184,
    identity_store_path: None,
};
run_agent(config).await
